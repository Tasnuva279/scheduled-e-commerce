FrischMarkt is a concept product designed to simplify how people receive fresh groceries from local markets. Instead of placing manual orders every week, users can subscribe to fresh product bundles and schedule automatic deliveries or pickups based on their preferred frequency.

The idea behind this product is to transform fresh grocery shopping from a repetitive ordering process into a predictable service experience. Once users define their schedule, the system automatically generates recurring deliveries until the user decides to modify or pause the service.

This repository contains the prototype screens and product concept for the FrischMarkt platform.
This repository includes:

Product concept and vision document in draft state

Figma prototype screens of the application flow

Early design exploration for the scheduling experience
